# Inteligencia Artificial y Machine Learning

## ¿Qué es la Inteligencia Artificial (IA)?
La **IA** es una tecnología que permite que las computadoras realicen tareas que normalmente requieren **inteligencia humana**, como reconocer imágenes, entender texto o tomar decisiones.

## ¿Qué es el Machine Learning?
El **Machine Learning (aprendizaje automático)** es una parte de la IA que permite que las computadoras **aprendan a partir de datos** y mejoren sin ser programadas paso a paso.

## Tipos de Machine Learning

### 1. Aprendizaje Supervisado
El modelo aprende con **datos que ya tienen la respuesta correcta**.

### 2. Aprendizaje No Supervisado
El modelo **encuentra patrones o grupos** en los datos sin respuestas previas.

### 3. Aprendizaje por Refuerzo
El modelo aprende **por prueba y error**, recibiendo recompensas o castigos según sus acciones.

## Resumen
- **IA:** máquinas que imitan inteligencia humana.  
- **Machine Learning:** aprendizaje a partir de datos.  
- **Tipos:** supervisado, no supervisado y por refuerzo.
